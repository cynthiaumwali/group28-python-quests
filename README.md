# Group 28 - Python Quests

A collaborative Python learning repository featuring 30 progressively challenging quests, from basic syntax to functions and mini-projects.

## Directory Structure

```
group28-python-quests/
├── quests/
│   ├── quest_01_first_spell.py
│   ├── quest_02_naming_ceremony.py
│   ├── quest_03_treasure_chest.py
│   ├── ... (and so on for all 30 quests)
│   └── quest_30_reflective_scribe.py
│
└── README.md
```

